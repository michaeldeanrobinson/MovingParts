# README #

This project demonstrates the basics for a WebApi2 application.

### Techniques and Patterns demonstrated ###

* Mediator
* Repository
* Factory
* Decorator
* AutoMapper
* FluentValidation
* Compression (including Brotli)

### How do I get set up? ###

* Clone repo
* Open in Visual Studio 2017
* Build
* Run
* Play around with Swagger
